This directory holds some extras for compiling with the NVIDIA Jetson TK1.

90-kinect2.rules is for the udev permissions

There are 2 install shell scripts:
- installLibfreenect2FromHere.sh can be used for installation libfreenect2 from the current project
- installLibfreenect2FromGithub.sh can be used by whose who just downloaded it.
  Note: You should also obtain installLibfreenect2ToDir.sh and place it near installLibfreenect2FromGithub.sh
  if you want to go by this way.
